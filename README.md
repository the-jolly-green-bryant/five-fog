# five-fog
Sample Ionic React Frontend w/ Static Page Rendering &amp; CI/CD

# Usage

## Initial Setup

```shell
pnpm install
```

## Running Dev Server

```shell
pnpm run dev
```

## Production Build

```shell
pnpm run build
```

# Target Features

- Index, scroll to paginate, filter
- View page
- Filter by click type
- CI/CD
- Static page rendering
- Accessibility feature
